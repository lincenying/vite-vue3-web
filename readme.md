# Vue3 web项目模板

## 已配置

```
Vue3、Vue-Router、Pinia, ESlint、Prettier、Element-plus、UnoCss
```

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm serve
```

### Compiles and minifies for production

```
pnpm build
```

### Preview for production

```
pnpm start
```

### Lints and fixes files

```
pnpm lint
```
